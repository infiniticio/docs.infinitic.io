When running, a workflow instance has a `context` property containing the following data:

| Name            | Type            | Description                                                                              |
| --------------- | --------------- | ---------------------------------------------------------------------------------------- |
| `id`            | UUID            | id of the workflow                                                                       |
| `tags`          | Set\<String\>   | tags of the workflow                                                                     |
